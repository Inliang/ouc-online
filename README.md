# OUC Online Learning Assistant 用户脚本文档

## 核心功能
- 视频自动播放，速度控制在0.5x-3.0x之间
- 自动回复讨论
- 自动查看材料
- 自动导航
- 页面类型识别

## 用户界面
- **左侧功能面板**：带状态指示器和速度控制
- **右侧日志面板**：显示实时日志
- 可拖动和最小化的面板

## 配置选项
- **USER_CONFIG**：包含视频速度和自动下一个延迟的设置

## 安装方法（Tampermonkey/Greasemonkey）
1. 安装 Tampermonkey 或 Greasemonkey 扩展
2. 添加新脚本，将脚本代码粘贴进去
3. 保存并启用脚本

## 使用场景
- 观看视频
- 完成讨论
- 查看材料

## 暂停功能
- 用户可以随时暂停脚本，便于中断学习

## 倍速调节
- 支持在0.5x至3.0x的范围内自由调节视频播放速度

## 日志类型表
| 日志类型 | 描述 |
|----------|------|
| success  | 操作成功 |
| warning  | 警告信息 |
| error    | 操作失败 |
| paused   | 脚本已暂停 |
